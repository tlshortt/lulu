<task type="auto">
  <name>Task 2: Configure Tailwind CSS and shadcn-svelte</name>
  <files>tailwind.config.js, src/app.css, src/routes/+layout.svelte</files>
  <action>
1. Install Tailwind CSS v4:
   ```bash
   npm install -D tailwindcss @tailwindcss/vite
   ```
2. Add to vite.config.ts:
   ```ts
   import tailwindcss from '@tailwindcss/vite';
   export default { plugins: [tailwindcss(), sveltekit()] };
   ```
3. Create src/app.css with Tailwind imports and dark theme:
   ```css
   @import "tailwindcss";
   @theme inline {
     --color-background: oklch(0.145 0 0);
     --color-foreground: oklch(0.92 0 0);
     --color-sidebar: oklch(0.17 0 0);
     --color-primary: oklch(0.92 0.2 250);
   }
   ```
4. Initialize shadcn-svelte:
   ```bash
   npx shadcn-svelte@next init --style default --colors slate
   ```
5. Add basic components: button, card, scroll-area
  </action>
  <verify>Run `npm run dev` - Tailwind classes work, shadcn components render</verify>
  <done>Dark theme applied, shadcn components available</done>
</task>
